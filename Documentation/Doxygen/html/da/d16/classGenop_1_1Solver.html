Opis szczegółowy

Klasa Solver implementuje algorytm Rungego-Kutty dla obiektu silnika DC:

x1 - prąd wirnika, x2 - prędkość kątowa
x1' = -(Ra / La) * x1 - (Gaf / La) * x2 + (1 / La) * U
x2' = (Gaf / J) * x1 - (B / J) * x2 + (1 / J) * Tl

Wersja
Wersja alfa.
